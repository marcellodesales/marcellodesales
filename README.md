## Programoji

Programoji is not an official or widely-adopted standard like Gitmoji, so it does not have associated GitHub emojis. However, we can provide a modified table that replaces Programoji emojis with GitHub emojis that have similar meanings:

| Emoji | Description | GitHub Code |
|-------|-------------|-------------|
| 🐛 | Bug | `:bug:` |
| 💻 | Code | `:computer:` |
| 📚 | Documentation | `:books:` |
| 🚀 | Deployment | `:rocket:` |
| 💡 | Idea | `:bulb:` |
| 🛠️ | In Progress | `:hammer_and_wrench:` |
| 💡 | Solution | `:bulb:` |
| 🧪 | Testing | `:test_tube:` |
| 📖 | User Story | `:book:` |
| 🚢 | Release | `:ship:` |
| 📝 | Release Notes | `:pencil:` |

Note that these GitHub emojis are not a direct replacement for Programoji emojis, and that their meanings may vary depending on the context of their use. Also note that the last GitHub emoji `:pencil:` is used for documentation and release notes interchangeably, while in Practice Gitmojii of using 📝 mainly for documentation and 📄 for release notes as I provided earlier.

Note that the tables for Codepoints and Gitmoji-cz only include a subset of the available emojis. The full list of relevant emojis for Codepoints can be found on their website, while Gitmoji-cz only includes a selection of Gitmoji emojis. The tables for emoji-commit and Programoji include some of the most commonly used emojis associated with software engineering.
